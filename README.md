# notion-my-crypto-wealth

Define in your env

```
NOTION_TOKEN=...
COINBASE_API_KEY=...
COINBASE_API_SECRET=...
HOLDING_DATABASE_ID=...
WEALTH_DATABASE_ID=...
```

Use node.js 14

```
nvm use 14
npm i
npm run dev
```
